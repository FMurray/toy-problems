# Triple Steps

A child is running up a staircase with N steps, and can either take it 1, 2, or 3 steps at a time.
Count how many ways the child can go up the stairs. 

## Input
N = number of steps

## Output
Integer representing the count of possible ways


## Examples

### Input 1

N = 4

### Output 1

1,1,1,1 -> 4
